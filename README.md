## What's in this repo

| File | Description |
| :--- | :--- |
| `index.html` | Minimal HTML page that loads lobster.js from CDN |
| `content.md` | Sample Markdown using lobster.js extended syntax |
| `server.js` | Zero-dependency static file server (Node.js built-ins only) |

## Run locally

```sh
npm start
```

Then open [http://localhost:3000](http://localhost:3000).
